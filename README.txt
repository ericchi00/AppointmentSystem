Appointment System
- Application allows the user to login and create appointments and customers. The application also provides validation to ensure appointments 
  are within business hours and also alerts user if there is an appointment within 15 minutes of login.

Author: Eric Chi
Contact Information: echi3@wgu.edu
Version: 1.0.0
Date: 11/28/2022

IDE Version: IntelliJ IDEA 2021.1.3 (Community Edition) Build #IC-211.7628.21, built on June 30, 2021
JDK Version: 17.0.5 (SDK Default)
JavaFX Version: JavaFX SDK 17.0.2

Directions on how to run:
Application can be run by running the main.class file or by running it directly in your IDE. User will first need to login before being able to access the main application menu.

Additional Report:
A report with the total count of specific locations in the appointments. (e.g. France - 2 or Conference Room - 3).
